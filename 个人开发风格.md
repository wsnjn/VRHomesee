# 工业极简风 (Industrial Minimal) - 开发规范

## 1. 核心理念 (Core Philosophy)

> **“工具属性优先于装饰属性”**

*   **理性 (Rational)**: 摒弃不必要的装饰、阴影和渐变，还原数据最本真的展示。
*   **精密 (Precision)**: 像素级的对齐，清晰的边界，像工业图纸一样严谨。
*   **高效 (Efficiency)**: 信息密度适中，通过高对比度和明确的网格快速传达信息。

---

## 2. 视觉法则 (Visual Rules)

### 🧱 "直角定律" (The Law of Squares)
**默认规则**：除特定“人”的元素外，UI 中**一切元素**均为严格直角。
*   **容器/卡片**: `border-radius: 0`
*   **按钮**: `border-radius: 0`
*   **输入框**: `border-radius: 0`
*   **标签/徽章**: `border-radius: 0`
*   **模态框**: `border-radius: 0`

### ⭕ "圆形例外" (The Circle Exception)
**唯一例外**：为了在视觉上快速区分“生物/人”与“无机/机器”，以下元素必须为正圆：
*   **用户头像 (Avatars)**: `border-radius: 50%`
*   **状态指示点 (Status Dots)**: `border-radius: 50%`

---

## 3. 色彩系统 (Color System)

整体采用低饱和度、冷色调的 **黑白灰 (Monochrome)** 体系，辅以极少量的功能色。

### 基础色 (Base)
| 角色 | 色值 | 说明 |
| :--- | :--- | :--- |
| **画布底色** | `#FFFFFF` | 纯白背景，保持干净 |
| **深色底/强强调** | `#111827` | (Tailwind Gray-900) 用于侧边栏背景、主按钮、强调边框 |
| **浅灰底** | `#F9FAFB` | (Gray-50) 用于列表交替、Hover背景 |
| **容器边框** | `#E5E7EB` | (Gray-200) 用于常规分割线 |

### 文字色 (Typography)
| 角色 | 色值 | 说明 |
| :--- | :--- | :--- |
| **主标题/正文** | `#111827` | 最深灰，接近纯黑 |
| **次级信息** | `#6B7280` | (Gray-500) 用于描述、图标 |
| **弱信息/占位** | `#9CA3AF` | (Gray-400) 用于时间戳、无关紧要信息 |

### 功能色 (Functional)
| 角色 | 色值 | 说明 |
| :--- | :--- | :--- |
| **在线/成功** | `#10B981` | (Emerald-500) 状态点 |
| **错误/删除** | `#EF4444` | (Red-500) 警示操作 |
| **自己消息泡** | `#4B5563` | (Gray-600) **特殊规定**：聊天中已发消息背景色 |

---

## 4. 排版与字体 (Typography)

*   **UI 字体**: `Inter`, system-ui, sans-serif
    *   强调清晰易读，字重以 `400` (Regular) 和 `600/700` (SemiBold/Bold) 为主，避免使用过细或过粗的字重。
*   **数据/代码字体**: `JetBrains Mono`, monospace
    *   用于：时间戳、ID、代码块、金额数字。

---

## 5. 组件规范 (Component Patterns)

### 按钮 (Buttons)
*   **样式**: 纯文本或带图标的矩形。
*   **主按钮**: 黑底白字 (`#111827`)。
*   **次级/图标按钮**: 无背景，Hover 时显现浅灰背景。
*   **交互**: Hover 变色响应迅速，无位移，无阴影扩散。

### 输入框 (Inputs)
*   **样式**: 只有边框，无背景色或极淡背景。
*   **Focus**: 边框变黑 (`#111827`)，无 `box-shadow` 光晕。

### 列表与表格 (Lists & Grids)
*   **分割**: 使用 `1px solid #E5E7EB` 实线分割。
*   **Hover**: 整行/整块变色 (`#F9FAFB`)。
*   **选中态**: 左侧出现粗黑竖线 `border-left: 3px solid #111827` 或背景加深。

### 聊天界面 (Chat Interface)
*   **气泡**: 直角矩形块。
*   **对方消息**: 浅灰背景 (`#F3F4F6`) + 深灰字。
*   **自己消息**: 中灰背景 (`#4B5563`) + 白字 (为了降低视觉刺眼度，不使用纯黑)。

---

## 6. 禁止事项 (Don'ts)

*   ❌ **禁止阴影**: `box-shadow: none` 是原则。
*   ❌ **禁止渐变**: 除非是展示图片本身，否则 UI 元素不使用渐变。
*   ❌ **禁止圆角容器**: 卡片、弹窗绝对不要圆角。
