# 个人开发风格指南

## 🧭 整体风格定位

**理性 / 冷静 / 工程感 / 长时间可用**

可以把它理解成：
- 👉 "工程师自用工具界面 + 少量编辑设计感"
- 👉 像 Linux 工具、数据库客户端、老牌技术站点那一挂

---

## 🎨 配色（低饱和 · 冷色 · 无情绪）

**主色调：冷灰 + 微蓝（接近工业色）**

### 背景
| 用途 | 色值 |
|------|------|
| 深冷黑（不是纯黑） | `#0E1116` |
| 次级背景 | `#161A22` |

### 文字
| 用途 | 色值 |
|------|------|
| 主文字 | `#C9CCD6` |
| 次级文字 | `#8B90A0` |

### 强调（极少用）
| 用途 | 色值 |
|------|------|
| 冷蓝（功能性，不装饰） | `#3A6EA5` |
| 中性灰 | `#6B7280` |

### ❌ 不用
- 紫色 / 渐变
- 高饱和强调色
- 情绪化色彩（橙 / 粉 / 暖黄）

### ✅ 原则
- 90% 灰阶
- 10% 功能色

---

## 🧱 形态规则（重点）

### 1️⃣ 全部直角
```css
border-radius: 0;
```

### 2️⃣ 不要阴影、不漂浮
```css
box-shadow: none;
```

用 **线条 / 间距** 代替层级。

---

## 🧩 布局逻辑（反 AI 的关键）

> AI 风 = 居中大块卡片 + 呼吸感过度

你要的是：

### ✅ 左对齐 + 网格感
- 强左对齐
- 固定列宽
- 明确分区

### 示意
```
│ NAV │ 内容内容内容内容内容
│     │ 内容内容内容内容内容
│     │ 内容内容内容内容内容
```

---

## ✍️ 字体（很重要）

### 首选（极推荐）
- Inter
- IBM Plex Sans
- Source Sans 3

### 可混搭
| 用途 | 字体 |
|------|------|
| 正文 | Inter |
| 数字 / 代码 | JetBrains Mono |

### 字重
```css
font-weight: 400; /* 或 500 为主 */
```

### ❌ 不要
- 夸张字重（禁止 300 / 600+）
- 圆润字体
- 手写 / 装饰字体

---

## 🧪 组件风格（像工具）

### 按钮
```css
background: transparent;
border: 1px solid #2A2F3A;
color: #C9CCD6;
```

**Hover 只是：**
```css
background: #1C2230;
```

### 输入框
- 明确边框
- 聚焦只是边框变亮
- 不发光、不变色

---

## 📐 间距系统（克制）

```css
/* 只用这三个 */
8px / 16px / 24px
```

### ❌ 不要
- 32 / 40 / 64 这种"设计师味"
