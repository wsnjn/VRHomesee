# 前端技术文档

本项目前端基于 **Vue 3** + **Vite** 构建，采用 **Industrial Minimal (工业极简)** 设计风格。集成 **Three.js** 实现 3D 全景看房与虚拟交互，使用 **ECharts** 展示数据大屏。

## 1. 技术栈 (Tech Stack)

| 类别           | 技术 / 库      | 版本     | 说明                                                          |
| :------------- | :------------- | :------- | :------------------------------------------------------------ |
| **核心框架**   | Vue 3          | ^3.5.22  | 使用 Composition API (`<script setup>`)                       |
| **构建工具**   | Vite           | ^7.1.11  | 极速冷启动与模块热更新 (HMR)                                  |
| **路由管理**   | Vue Router     | ^4.6.3   | 单页应用 (SPA) 路由控制                                       |
| **3D 引擎**    | Three.js       | ^0.181.1 | 实现 VR 全景看房、3D 模型展示、AR 手势交互                    |
| **数据可视化** | ECharts        | ^6.0.0   | 房东/管理员仪表盘数据图表                                     |
| **HTTP 请求**  | Axios          | ^1.13.2  | 与 Spring Boot 后端进行 RESTful API 通信                      |
| **状态管理**   | Reactive State | -        | 基于 Vue 3 `reactive/ref` 的轻量级状态管理 (位于 `src/state`) |

## 2. 项目目录结构

```
Vue3/frontend
├── public/                 # 静态资源目录 (不经 Vite 打包)

│   ├── assets/             # 静态资源 (图片、CSS)
│   ├── components/         # 公共组件 (Navbar, Cards, Modals)

│   ├── views/              # 页面级组件 (Pages)
│   │   ├── admin/          # 管理员后台 (用户管理、社区治理)

│   │   ├── tenant/         # 租客门户 (找房、我的合同、报修)
│   │   ├── Login.vue       # 统一登录页
│   │   ├── VirtualWorld.vue # 3D 虚拟看房核心页面

└── vite.config.js          # Vite 构建配置
```

## 3. 核心功能模块

### 3.1 角色工作台系统
前端通过路由守卫和布局区分不同角色的操作界面：
*   **房东 (Landlord)**: 极简仪表盘，管理房源状态 (上架/发布)、处理预约、查看收益图表 (ECharts)。
*   **租客 (Tenant)**: 瀑布流房源列表、高级筛选、个人中心 (合同/报修)。
*   **管理员 (Admin)**: 全局用户管控、社区内容审核、系统数据监控。

### 3.2 3D 与 VR 交互 (`VirtualWorld.vue`)
*   利用 **Three.js** 构建沉浸式看房体验。
*   支持全景图 (360 Image) 加载与场景切换。
*   实现空间坐标点的点击交互 (Hotspots)。

### 3.3 社区系统 (`views/community`)
*   **动态流**: 仿社交媒体的图文动态展示，支持点赞与评论。
*   **即时通讯**: 模拟聊天窗口界面，支持私聊与群聊 UI。

## 4. 安装与运行

确保已安装 Node.js (推荐 v18+)。

```bash
# 1. 进入前端目录
cd Vue3/frontend

# 2. 安装依赖
npm install

# 3. 启动开发服务器 (默认端口 5173)
npm run dev

# 4. 构建生产环境代码
npm run build
```

## 5. 开发规范
*   **文件命名**: 组件使用大驼峰 (PascalCase)，如 `HouseList.vue`。
*   **样式**: 推荐使用 Scoped CSS，遵循 "Industrial Minimal" 设计原则 (直角、单色、无多余阴影)。
*   **API 调用**: 统一封装在组件内部或独立 Service 文件中，使用 Axios 拦截器处理 Token。
